# Pytinhas

Projeto desenvolvido com o intuito de contribuir como equipe de desenvolvedores, em um projeto com dados reais, arquiteturas, metodologias/ferramentas ageis.

## Proposito do projeto

Desenvolver uma estufa de plantas automatizado, gerindo agua, luz, temperatuda, humidade e ventilação.

ESP-8266 é um dispositivo que se conecta na rede wi-fi e tem como finalidade o envio de informações no processo de desenvolvimento da planta, sendo possivel diagnosticar a humidade, temperatura e quantidade de agua no solo.

## Componetes usados

ESP-8266, DHT11, Relay 5V 2 Canais, Micro Bomba De Água, Display 16x2 e Módulo Sensor De Umidade De Solo.
